# Learn Finesse - Global Finishing School

**Learn Finesse** is your digital "global finishing school" for mastering modern, cross-cultural etiquette, soft skills, and professional polish.

## Overview

Designed for ambitious individuals and global talent, this app delivers an immersive 10-Day Bootcamp across Western, Indian, and Eastern contextsâ€”bridging knowledge with global industry demands.

## Curriculum Structure

### 5 Core Modules:

1. **The Internal Blueprint:** Personal branding, grooming, digital image
2. **The Communication Suite:** Verbal artistry, active listening, EQ
3. **The Global Table & Social Grace:** Dining etiquette & networking across three regions
4. **The Career Accelerator:** STAR/STAR-P interviews, resumes, managing up
5. **The Modern Finesse Factor:** Digital ethics, AI etiquette, resilience

Each module incorporates Western, Indian, and Eastern perspectives.

## Features

- **Interactive Cultural Pivots:** Real-world scenarios with branching logic
- **Visual Learning:** Bow angles, table settings, greeting protocols
- **Skill Badges:** Track progress without formal certification
- **Actionable Missions:** Daily real-world practice tasks
- **Global Pivot Tables:** Side-by-side cultural comparisons

## Development

```bash
# Install dependencies
yarn install

# Run development server
yarn dev

# Build for production
yarn build

# Start production server
yarn start
```

The app runs on port **3025** in development.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Framer Motion (animations)
- Supabase (authentication & data)
